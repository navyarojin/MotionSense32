# MotionSense32
MotionSense-32 is a compact 2-layer ESP32-based development board with an integrated battery charging system and 6-axis IMU for motion tracking applications.

## Features

* ESP32-S3 Mini module
* 6-axis IMU (Accelerometer + Gyroscope)
* Onboard LiPo battery charger
* USB interface for programming and power
* Low dropout (LDO) voltage regulator
* Compact 2-layer PCB design

## Power System

* USB powered
* 3.7V LiPo battery support
* Integrated charging IC

## IMU

* 6-axis motion tracking
* I2C interface

## Hardware Files

All design files including schematics, PCB layout, and Gerbers are available in the `/Hardware` folder.
